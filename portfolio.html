Fizz-Buzz

A project to program a simple counting game. Fizz-Buzz is a simple game to teach division to kids. Enter any number and the program will count out the numbers for you. It will return "Fizz" for numbers divisible by 3 and "Buzz" for numbers divisible by 5. This taught me some of the basic tools of programming like for-loops and if/else statements.
